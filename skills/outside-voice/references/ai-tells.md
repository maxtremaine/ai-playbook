# AI tells to remove

A catalog of the phrases, structures, and sentence patterns that mark prose as machine-generated. Adapted from Stop Slop by Hardik Pandya (github.com/hardikpandya/stop-slop, MIT License).

## Calibration for Sherpa (read first)

These rules subtract. They tell you what to cut, not what to write. Sherpa's voice tells you what to write: warm, guiding, clear, optimistic.

Two failure modes, not one:
- **Over-written slop.** The patterns below. Fix by cutting.
- **Over-corrected slop.** Stripping so hard the prose goes cold, clipped, and choppy. Fragments stacked for effect, every adverb gone, every sentence the same short length. This reads as machine-generated too, and it kills Sherpa's warmth.

So:
- Cut adverbs that add nothing. Keep adverbs that carry real meaning and fit an empathetic tone. "We know visas can be confusing" keeps "can." "We're really thrilled" loses "really."
- Vary sentence length the way speech does. Do not flatten every line to a fragment.
- Compress for clarity, not for drama. "Move faster. Your competition is." wins a contest and loses the brand. Sherpa would write a warmer, clearer version.
- When a rule here fights Sherpa's voice, strip the empty version and keep the human one.

## Throat-clearing openers

Announcement phrases that delay the point. State the content directly.

- "Here's the thing:"
- "Here's what / this / that / why [X]"
- "The uncomfortable truth is"
- "It turns out"
- "The real [X] is"
- "Let me be clear"
- "The truth is,"
- "I'm going to be honest"
- "Can we talk about"
- "Here's what I find interesting"
- "Here's the problem though"

Any "here's what / this / that" construction is throat-clearing. Cut it and state the point.

## Emphasis crutches

These add no meaning. Delete them.

- "Full stop." / "Period."
- "Let that sink in."
- "This matters because"
- "Make no mistake"
- "Here's why that matters"

## Business jargon

Replace with plain language. Sherpa is educational, not corporate.

| Avoid                 | Use instead            |
|:--------------------- |:---------------------- |
| Navigate (challenges) | Handle, address        |
| Unpack (analysis)     | Explain, examine       |
| Lean into             | Accept, embrace        |
| Landscape (context)   | Situation, field       |
| Game-changer          | Significant, important |
| Double down           | Commit, increase       |
| Deep dive             | Analysis, examination  |
| Take a step back      | Reconsider             |
| Moving forward        | Next, from now         |
| Circle back           | Return to, revisit     |
| On the same page      | Aligned, agreed        |

## Adverbs

Cut adverbs that only add emphasis. The usual empty offenders: really, just, literally, genuinely, honestly, simply, actually, deeply, truly, fundamentally, inherently, inevitably, interestingly, importantly, crucially.

Keep an adverb only when it changes the meaning of the sentence. Test: delete it and read the sentence. If nothing is lost, it stays deleted.

Also cut these filler phrases:
- "At its core"
- "In today's [X]"
- "It's worth noting"
- "At the end of the day"
- "When it comes to"
- "In a world where"
- "The reality is"

## Meta-commentary

Remove self-referential asides. The piece should move, not narrate its own structure.

- "Hint:" / "Plot twist:" / "Spoiler:"
- "You already know this, but"
- "But that's another post"
- "X is a feature, not a bug"
- "The rest of this essay explains..."
- "Let me walk you through..."
- "In this section, we'll..."
- "As we'll see..."
- "I want to explore..."

## Performative emphasis and false intimacy

- "creeps in"
- "I promise" / "They exist, I promise"
- Manufactured sincerity in general

## Telling instead of showing

Announcing difficulty or significance instead of demonstrating it.

- "This is genuinely hard"
- "This is what leadership actually looks like"
- "This is what [X] actually looks like"
- "actually matters"

## Vague declaratives

Sentences that announce importance without naming the specific thing. Cut them or replace with the specific thing.

- "The reasons are structural"
- "The implications are significant"
- "This is the deepest problem"
- "The stakes are high"
- "The consequences are real"

## Binary contrasts

False drama through telegraphed reversal. State the point directly.

| Pattern                             | Problem                  |
|:----------------------------------- |:------------------------ |
| "Not because X. Because Y."         | Telegraphed reversal     |
| "[X] isn't the problem. [Y] is."    | Formulaic reframe        |
| "The answer isn't X. It's Y."       | Predictable pivot        |
| "It feels like X. It's actually Y." | Setup and reveal cliche  |
| "Not X. But Y." / "not X, it's Y"   | Mechanical contrast      |
| "stops being X and starts being Y"  | False transformation arc |
| "is about X but not Y"              | False distinction        |
| "not just X but also Y"             | Additive hedge           |

Instead: state Y directly. Drop the negation.

## Negative listing

Listing what something is not before revealing what it is. State the thing.

- "Not a X... Not a Y... A Z."
- "It wasn't X. It wasn't Y. It was Z."

## Dramatic fragmentation

Sentence fragments for emphasis read as manufactured profundity. Use complete sentences.

- "[Noun]. That's it. That's the [thing]."
- "X. And Y. And Z."
- "This unlocks something. [Word]."

## Rhetorical setups

These announce insight instead of delivering it. Make the point and let the reader draw the conclusion.

- "What if [reframe]?"
- "Here's what I mean:"
- "Think about it:"
- "And that's okay."

## False agency

Inanimate things given human verbs. AI loves this because it avoids naming the actor. Name the person, or use "you."

| Pattern                     | Fix                            |
|:--------------------------- |:------------------------------ |
| "a complaint becomes a fix" | Someone fixed it               |
| "the decision emerges"      | Someone decides                |
| "the culture shifts"        | People change behavior         |
| "the data tells us"         | Someone reads it and concludes |
| "the market rewards"        | Buyers pay for things          |

This one matters for Sherpa specifically. "Sherpa turns a compliance burden into revenue" is fine as a claim, but watch for the lazy version where requirements, borders, or systems act on their own. People act. Airlines sell. Travelers apply.

## Narrator-from-a-distance

Floating above the scene instead of putting the reader in it. Sherpa puts the traveler or partner in the room.

- "Nobody designed this."
- "This happens because..."
- "People tend to..."

Instead: "You don't sit down one day and decide to..." beats "Nobody designed this."

## Passive voice

Every sentence needs a subject doing something. Passive voice hides the actor.

| Pattern                    | Fix                  |
|:-------------------------- |:-------------------- |
| "X was created"            | Name who created it  |
| "It is believed that"      | Name who believes it |
| "The decision was reached" | Name who decided     |

This is also Sherpa's first tone rule. "Yana sent the forms," not "the forms were sent."

## Sentence starters to avoid

| Pattern                                               | Fix                                                       |
|:----------------------------------------------------- |:--------------------------------------------------------- |
| Starting with What, When, Where, Which, Who, Why, How | Lead with the subject or verb, or name the specific thing |
| Paragraphs starting with "So"                         | Start with content                                        |
| Sentences starting with "Look,"                       | Remove                                                    |

"What makes this hard is..." becomes "The constraint is..." or, better, name the constraint.

## Rhythm

| Pattern                               | Fix                                                                     |
|:------------------------------------- |:----------------------------------------------------------------------- |
| Three-item lists by default           | Two items often read better. Use three when you have three real things. |
| Questions answered immediately        | Let a question breathe, or cut it                                       |
| Every paragraph ends on a punchy line | Vary the endings                                                        |
| Em dashes                             | Remove. Comma, colon, semicolon, or two sentences.                      |
| Stacked short punchy sentences        | Vary length. See the calibration note.                                  |
| "Not always. Not perfectly."          | Hedging disguised as reassurance                                        |

## Word patterns

| Pattern                                                                  | Problem                                  |
|:------------------------------------------------------------------------ |:---------------------------------------- |
| Lazy extremes: every, always, never, everyone, everybody, nobody         | False authority. Use specifics.          |
| Empty intensifiers: really, just, literally, genuinely, simply, actually | Empty emphasis. See the adverbs section. |

## Quick checks before delivering

- Adverb that adds nothing? Cut it.
- Passive voice or inanimate actor? Name who acts.
- Wh- sentence opener? Restructure.
- "Here's what / this / that" throat-clearing? Cut to the point.
- "Not X, it's Y" contrast? State Y.
- Three sentences the same length? Break one. But do not chop everything to fragments.
- Paragraph ends on a one-liner every time? Vary it.
- Em dash anywhere? Remove it.
- Vague declarative? Name the specific thing.
- Did the edit make it cold? Add the warmth back.
